# Thermal Printer Web UI

[English](./README.md) | 日本語

ポータブルサーマルプリンター向けに設計された、モダンなウェブベースのラベル・写真印刷アプリケーション。Web Bluetooth経由でデバイスに接続し、ブラウザから直接カスタムラベルの作成や高品質な写真印刷が可能です。

### 🌐 ライブデモ (Web UI)
**[Thermal Printer Web UI を開く](https://thermal-printer-web-ui-1.onrender.com/)**

> [!NOTE]
> - **テスト用**: このライブデモはテスト目的で提供されています。アクセスが集中すると、一時的に利用できなくなったり、動作が重くなったりする場合があります。
> - **プライバシーとデータ**: このアプリケーションはクライアントサイドで動作します。ユーザーのデータ（ラベル、写真、設定）が**サーバーに保存されることはありません**。すべてお使いのデバイスのブラウザ内で処理・保存されます。

## 機能

### 🏷️ プロフェッショナル・ラベルメーカー
細部までこだわったカスタムラベルをデザイン：
- **スマートサイジング**:
    - **自動縮小 (Auto-Shrink)**: 長いテキストがラベル幅に完璧に収まるよう、フォントサイズを自動的に調整します。
    - **自動拡大 (Auto-Expand)**: 空白を埋めて力強い印象を与えるよう、フォントサイズを最大限まで拡大します。
- **テキスト配置**: 左寄せ、中央揃え、右寄せから選択可能。
- **フォントのカスタマイズ**: 様々なフォントファミリーから選択し、必要に応じてフォントサイズを手動で調整できます。
- **長さ制御**: ラベルの長さを「自動」（テキストに応じて動的に変化）または固定サイズ（例：28mm）に設定可能。
- **印刷方向**: 横書きと縦書きの両方に対応。

### 🖼️ 高度な写真印刷
サーマルペーパーで最高の印刷品質を実現するために画像を最適化：
- **自動スケーリング**: プリンターの最大幅（通常384px）に合わせて画像を自動的にリサイズします。
- **自動トリミング (Auto Trim)**: 白地や透明な余白をインテリジェントに検出し、紙を節約しながら被写体にフォーカスします。
- **画像処理メソッド**:
    - **ディザリング (Floyd-Steinberg)**: 写真や複雑な画像に最適で、網点パターンを使用して滑らかなグラデーションを作成します。
    - **しきい値判定 (Black & White)**: ロゴやQRコード、シンプルなグラフィックに最適で、鮮明でコントラストの高い結果が得られます。

### 📊 Excel一括印刷
スプレッドシートのデータを使用して、複数のラベルを一度に効率的に印刷：
- **一括アップロード**: `.xlsx` および `.xls` ファイルをサポートし、ラベル作成を自動化します。
- **高度な行処理**:
    - **ループ対応**: Excelファイルに「Loop」列を追加することで、同じ行を複数回印刷できます。
    - **セグメント分割**: セル内でセミコロン (`;`) を使用することで、セグメントごとに個別のラベルを生成できます。
- **フォーマットオプション**:
    - **コンパクトモード**: キーと値をインテリジェントにまとめてスペースを最適化します。
    - **自動折り返し**: 長いデータ入力が384px幅の中で正しく折り返されるようにします。
- **精密なバッチ制御**:
    - **印刷遅延**: ラベル間のカスタム遅延（秒）を設定し、プリンターの冷却や取り扱いを容易にします。
    - **送り量設定**: ラベル間の余白（mm）を設定し、切り離しやすくします。
- **リアルタイムプレビュー**: 印刷ジョブを開始する前に、生成されたすべてのラベルをスクロール可能なリストで確認できます。

### ⚡ スマートな接続性とUI
- **Web Bluetooth**: ブラウザからプリンターへ直接接続。ドライバーや追加ソフトは不要です。
- **多言語対応**: 英語と日本語に完全対応。
- **レスポンシブデザイン**: デスクトップとモバイル両方のブラウザに最適化された、プレミアムでモダンなインターフェース。

### ⚙️ 詳細設定
コントロール下部の **「詳細設定 (Detailed Options)」** メニューから、より細かな調整が可能です：
- **ブラウザのスリープ防止**: 長時間の使用中も画面を点灯させ続けます（Wake Lock APIのサポートが必要）。
- **プリンターのスリープ防止**: プリンターの自動電源オフを回避するためにキープアライブ信号を送信します。
- **拡張フォントサイズ**: より細かいフォントサイズ選択が可能になります。
- **拡張フォント**: より多くのシステムフォントファミリーを選択できるようになります。
- **Excelモード**: スプレッドシートデータの一括処理用インターフェースに切り替えます。
- **印刷方向**: **横書き** または **縦書き** の印刷方向を選択できます。

## 始め方

### 前提条件
- 対応するサーマルプリンター（LX-D02など）。
- Web Bluetoothをサポートするブラウザ（Chrome、Edge、Operaを推奨）。
- ローカル開発用の **Node.js** および **npm**。

### インストール
1. リポジトリをクローン：
   ```bash
   git clone https://github.com/your-username/lxprint.git
   cd lxprint
   ```
2. 依存関係をインストール：
   ```bash
   npm install
   ```
3. 開発サーバーを起動：
   ```bash
   npm run dev
   ```
4. ブラウザで表示されたローカルURLを開きます。

### 使い方

#### 1. 接続
ヘッダーの **"Connect"** ボタンをクリックします。Bluetoothペアリングダイアログからプリンターを選択します（通常、名前は "LX-" で始まります）。

#### 2. ラベルメーカー
- テキストエリアに文字を入力します。プレビューはリアルタイムで更新されます。
- **自動縮小** または **自動拡大** を使用して、テキストをラベルに最適化します。
- ラベルの長さを固定、または「自動」に設定して動的に高さを調整します。
- **"Print Label"** をクリックしてプリンターに送信します。

#### 3. Excel一括印刷
- 詳細設定で **Excelモード** を有効にします。
- `.xlsx` または `.xls` ファイルをアップロードします。
- スクロール可能なプレビューリストでラベルの内容を確認します。
- **"Excel Print"** をクリックして一括印刷を開始します。進行状況（例：「1/50」）が画面に表示されます。

#### 4. 写真印刷
- **Photo** タブに切り替えます。
- 画像ファイルを選択します。プリンターの幅に合わせて自動的にスケーリングされます。
- **自動トリミング** を切り替えて、不要な余白を削除します。
- 好みの処理方法を選択します（写真には **ディザリング**、ロゴには **しきい値判定** が推奨されます）。
- **"Print Photo"** をクリックします。

## トラブルシューティング

**Q: プリンターが見つからない、または接続できません。まず何を確認すべきですか？**
- **Bluetoothの状態**: システムでBluetoothが有効になっていることを確認してください。
- **ブラウザの確認**: `chrome://bluetooth-internals/#devices` にアクセスして、ブラウザがデバイスを検出しているか確認してください。

**Q: Macのペアリングダイアログにプリンターが表示されません。**
- macOSの検出は不安定な場合があります。App Storeで入手できる **LightBlue** アプリを使用して、プリンター（通常は "LX-..."）が電波を発信しているか確認してください。

**Q: 接続が切れ、再接続できません。**
- Web Bluetoothがハングすることがあります。ブラウザを完全に終了（Macの場合は **Command + Q**）してBluetoothキャッシュをクリアしてから、もう一度試してください。
